# Technologie Internetowe — Projekty laboratoryjne

W tym repozytorium będą umieszczane wszystkie projekty laboratoryjne związane z przedmiotem *Technologie Internetowe*.  
Każde ćwiczenie znajduje się w osobnym katalogu (np. `Lab1`, `Lab2`, ...). Dla każdego zadania powinien znajdować się kod źródłowy oraz krótka instrukcja uruchomienia w pliku README znajdującym się w katalogu danego laboratorium.

Zasady:
- Każde laboratorium w osobnym folderze: `LabN/`.
- W folderze lab powinna być minimalna dokumentacja (cel, wymagania, sposób uruchomienia).
